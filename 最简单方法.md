# 最简单的方法 🎯

## 方法一：在 Termux 中一键安装（推荐）⭐

```bash
# 1. 将 Termux脚本 文件夹复制到手机
# 2. 在 Termux 中执行：
cd ~/Termux脚本
bash install-termux.sh
```

**就这么简单！**

---

## 方法二：如果网络有问题

```bash
# 在 Termux 中执行：
export GOPROXY="https://goproxy.cn,direct"
cd ~/Termux脚本
bash install-termux.sh
```

---

## 方法三：直接运行（不编译）

如果你不想编译，可以直接运行源码：

```bash
# 在 Termux 中：
pkg install golang
cd ~/Termux脚本
export GOPROXY="https://goproxy.cn,direct"
go run main.go
```

**注意**：每次运行都会编译，第一次会比较慢。

---

## 方法四：使用预编译文件

如果有人已经编译好了，直接下载使用：

```bash
# 下载后
chmod +x fatalder-termux
./fatalder-termux
```

---

## 总结

**最简单**：方法一（在 Termux 中一键安装）
**最快**：方法四（使用预编译文件）
**最灵活**：方法三（直接运行源码）
