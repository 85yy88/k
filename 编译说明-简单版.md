# 最简单的安装方法

## 在 Termux 中直接编译（最简单）

### 步骤：

1. **将 `Termux脚本` 文件夹复制到手机**

2. **在 Termux 中执行：**
   ```bash
   cd ~/Termux脚本
   bash install-termux.sh
   ```

3. **等待编译完成，然后运行：**
   ```bash
   ./fatalder-termux
   ```

就这么简单！

---

## 如果网络有问题

在 Termux 中执行：
```bash
export GOPROXY="https://goproxy.cn,direct"
bash install-termux.sh
```

---

## 如果还是不行

手动执行：
```bash
pkg install golang
cd ~/Termux脚本
export GOPROXY="https://goproxy.cn,direct"
go mod download
go build -ldflags="-s -w" -o fatalder-termux main.go
chmod +x fatalder-termux
./fatalder-termux
```
